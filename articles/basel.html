バーゼル問題の収束おっっっっそ!

有名な定理としてすべての自然数の2乗の逆数の和は円周率の二乗を6で割ったものと等しいというものがあるんですけど、それを円周率の計算に使えるかなと思ってみたので実装してみたところ、100万項まで計算しても小数点以下5桁までしか一致してなくて絶望しました。

コード(Python)

                
                    import math

                    def calculate_pi(n):
                        pi_sum = 0
                        for i in range(1, n+1):
                            pi_sum += 1/(i**2)
                        pi = math.sqrt(pi_sum*6)
                        return pi
                    
                        print(calculate_pi(1000000))

良い子のみんなはマチンの公式とかG-Lアルゴリズムとかを使おうね!
